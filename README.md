# Merge Function Project

This project implements a TypeScript function to merge three sorted arrays (one descending, two ascending) into a single sorted ascending array.

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

## Running Tests

To run the unit tests, execute:

```bash
npm test
```

## Project Structure

- `src/merge.ts`: Contains the implementation of the `merge` function.
- `tests/merge.test.ts`: Contains the unit tests using Jest.
- `tsconfig.json`: TypeScript configuration.
- `jest.config.js`: Jest configuration.
